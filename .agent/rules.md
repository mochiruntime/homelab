# Agent Rules

## Environment
- **Virtual Environment**: Always ensure the Python virtual environment (`.venv`) is activated before running commands.
- **Shell & Platform Awareness**: The environment may use **WSL** and / or the **fish** shell. Check the current environment and use the appropriate activation script (e.g., `source .venv/bin/activate.fish` for fish).

## YAML Styling
- **Preference**: Always use **Block Style** for YAML (indentation and dashes) instead of Flow Style (braces and brackets).
